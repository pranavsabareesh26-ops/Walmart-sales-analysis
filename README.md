# Walmart Sales Analysis

## Overview
This project analyzes Walmart sales data using Python and data visualization techniques to identify trends, seasonal patterns, and factors influencing weekly sales performance.

The project focuses on:
- Weekly sales trends
- Seasonal sales analysis
- Sales vs temperature relationship
- Sales vs unemployment analysis

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

---

## Dataset Information
The dataset contains Walmart sales records including:
- Weekly sales
- Temperature
- Unemployment rate
- Dates
- Store information

- Sales peaks are visible during holiday periods
